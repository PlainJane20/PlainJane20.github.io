# Navi Sohi - Professional Portfolio Website

Modern, colorful portfolio website showcasing professional experience, projects, certifications, and LinkedIn recommendations.

## 🌟 Features

- **Modern Design**: Gradient backgrounds, glassmorphism effects, smooth animations
- **Dark/Light Mode**: Toggle between themes
- **Mobile Responsive**: Fully responsive design
- **Interactive Elements**: Hover effects, scroll animations, 3D card effects
- **LinkedIn References**: Real recommendations from colleagues
- **Downloadable Resume**: One-click PDF download

## 📄 Adding Your Resume PDF

To enable the "Download Resume" button:

1. **Convert your resume to PDF** (recommended name: `NaviSohi_Resume.pdf`)
2. **Add the PDF to this directory** (same folder as index.html)
3. The download button is already configured to link to `NaviSohi_Resume.pdf`

### Alternative Options:

**Option 1: Different filename**
If your PDF has a different name, update line in `index.html`:
```html
<a href="YOUR_FILENAME.pdf" download class="btn btn-primary">Download Resume</a>
```

**Option 2: External Link (Google Drive, Dropbox, etc.)**
```html
<a href="https://drive.google.com/YOUR_LINK" target="_blank" class="btn btn-primary">View Resume</a>
```

**Option 3: Use your current PDF**
Copy your PDF from the current location:
```bash
cp NaviSohi_Google_Workspace_Resume.pdf NaviSohi_Resume.pdf
```
